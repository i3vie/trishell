# trishell
if you want to use the makefile for smallest possible releases ensure that `upx` is on your path
## support
### architectures
- x86_64
- aarch64
- riscv64<br>
it should probably be buildable on anything that rust can target
### features
#### builtins
- cd
- pwd
#### thing
- ~ as home dir
- env variable substitution
- configurable prompt
